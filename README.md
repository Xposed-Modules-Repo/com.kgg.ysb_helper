# 钥匙宝换机登录

使用该模块在新手机上可以直接登录钥匙宝，不需要找物业重新绑定，仅用于学习交流，请勿用于任何非法用途

# 更新日志

2023.05.07&nbsp;&nbsp;&nbsp;&nbsp;v1.0.0&nbsp;&nbsp;&nbsp;&nbsp;首次发布

# 使用说明

1.在旧手机上启动模块，读取手机型号和设备ID(旧手机无需安装LSPosed)

2.在新手机上启动模块，填入旧手机手机型号和设备ID并保存

3.重启钥匙宝进程，直接登录即可
